# **App Name**: CipherSite

## Core Features:

- Events Calendar: Display upcoming meeting dates, times, and locations.
- Newsletter Subscription: Allow members to subscribe to a newsletter for updates and announcements.
- News Feed: Display club news and announcements.
- Forums: Include an interactive forum where members can discuss topics and ask questions.
- Security News Summarizer: Summarize security news with a tool that reasons when information is safe or unsafe.

## Style Guidelines:

- Primary color: Electric blue (#7DF9FF) to convey technological sophistication and energy. 
- Background color: Dark navy (#1A237E), nearly the same hue as the primary color but heavily desaturated to around 20%.
- Accent color: A vibrant purple (#BE29EC) which provides a modern high-tech feel.
- Body and headline font: 'Space Grotesk', a modern sans-serif for clear readability and a futuristic look.
- Use streamlined, glowing icons that reflect cybersecurity themes.
- Maintain a modern layout with clean lines, generous spacing, and well-structured sections.
- Employ subtle transitions and interactive elements for a smooth, engaging user experience.
